# ML From Scratch

A collection of Jupyter notebooks implementing machine learning algorithms where I focus on the mathematics behind them rather than using a library like PyTorch.

## Topics Covered

- **Regression**: Linear and logistic regression with Gradient Descent
- **Tree-Based Models**: Decision trees
- **Dimensionality Reduction**: Principal Component Analysis
- **Clustering**: Unsupervised learning techniques like K-Means and DBSCAN
- **Neural Networks**: Fully connected and optimized Deep NNs (DNNs), Recurrent NNs (RNNs), and Convolutional NNs (CNNs)
- **Generative Models**: Generative Adversarial Networks (GANs) and Transformers
